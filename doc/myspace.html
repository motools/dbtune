DBTune.org
Myspace RDF Service

1. Introduction

This service provides a live RDF representation of a target Myspace page. If the page is also an "artist" page, then the tracks with availableAs links to the streaming audio cache are included in the RDF. Note that only the "top friends" in the target Mysapce page are included in the RDF.

This representations is designed using the Music Ontology and FOAF.

This effort is part of the Linking Open Data on the Semantic Web community project, hosted by the Semantic Web Education and Outreach interest group.

2. Using this service

Starting from a Myspace URL: 

http://myspace.com/<userURL>
Just get the following URI: 
http://dbtune.org/myspace/<userURL>
Alternatively, you can use the Myspace user id directly: 
http://dbtune.org/myspace/uid/<userID>

Try to load this URI in the Tabulator

3. Code
